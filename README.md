# Simulador-Espectros-FBG
App interactiva para el análisis de redes de Bragg. Permite importar datos reales, ajustar parámetros físicos ($L$, $\Lambda$, $\Delta n$) y optimizar espectros mediante lsqcurvefit. Incluye visualización lineal/logarítmica, cálculo de $R^2$ y herramientas de barrido paramétrico.

Funciones Clave:

Ajuste Automático: Optimización de parámetros físicos ($L$, $\Lambda$, $\Delta n$) mediante lsqcurvefit.

Visualización Dual: Gráficas en tiempo real en escala Lineal y Logarítmica (dB).

Análisis de Precisión: Cálculo instantáneo del coeficiente de correlación $R^2$.

Barrido Paramétrico: Herramienta para analizar la sensibilidad del espectro.

Importación Directa: Carga de datos experimentales desde archivos .txt.
